KEYBOARD SHORTCUTS

RECORDING
  Space        Toggle recording
  P            Play current recording
  S            Stop playback
  O            Toggle monitoring

SELECTION & PLAYBACK
  Click        Set position marker (cyan line)
  Drag         Create selection range
  Drag marker  Resize selection or move position marker
  P            Play from marker or play selection
  Escape       Clear marker/selection

AUDIO EDITING
  Ctrl+D       With selection: Delete selected range
  Space        With marker: Insert recording at position
  Space        With selection: Replace selection with recording
  Ctrl+Z       Undo last edit operation
  Shift+Ctrl+Z Redo last undone operation

NAVIGATION
  Up/Down      Navigate utterances
  Left/Right   Browse takes

FLAGGING
  Shift+E      Toggle 'Needs Edit' flag
  Shift+X      Toggle 'Rejected' flag
  Shift+U      Clear flag
  Ctrl+E       Jump to next 'Needs Edit'
  Ctrl+X       Jump to next 'Rejected'

DISPLAY
  M            Toggle mel spectrogram & level meter
  I            Toggle info overlay
  F10          Toggle fullscreen

SECOND WINDOW (Shift + Key)
  Shift+M      Toggle meters in second window
  Shift+I      Toggle info panel in second window
  Shift+F10    Toggle fullscreen in second window

FILE OPERATIONS
  Ctrl+D       Delete current recording

DIALOGS
  Ctrl+F       Find utterance
  Ctrl+U       Utterance order
  Ctrl+I       Session settings

SESSION MANAGEMENT
  Ctrl+N       New session
  Ctrl+O       Open session
  Ctrl+Q       Quit

HELP
  F1           Show keyboard shortcuts
